Here are a few useful websites that can help you develop your coding skills and discover new languages. If you have any tips for more sites we can include here feel to drop us an  email.

---

#### Beginner

[Scratch](http://scratch.mit.edu/) – Scratch is a visual programming environment and toolkit for making games, animated stories and interactive art. If you’ve never coded before this is a great place to start! It has a huge online community and loads of starter projects to get you up and running. Here’s a [link](https://resources.scratch.mit.edu/www/cards/en/Scratch2Cards.pdf) to the cards that we use.

[Alice](http://www.alice.org/) – Alice is kind of the 3D version of Scratch: you can make 3D animations, videos and interactive games using a graphical drag-and-drop interface. Alice teaches fundamental programming concepts through fun projects.

[Khan Academy](https://www.khanacademy.org/cs) – Khan Academy has a great Computer Science section that teaches a broad range of introductory programming concepts through easy-to-follow videos. The rest of the Khan Academy site also has loads of useful related tutorials in maths, science and engineering.

[Codecademy](http://www.codecademy.com/) –  a fun and interactive way to learn how to code, Codecademy lets you write code directly in your browser and learn a wide range of languages by working on real-world projects. Current courses include HTML & CSS, Javascript, PHP, Python and Ruby.

[Code School](https://www.codeschool.com/) – similar to Codecademy, Code School teaches web technologies in the comfort of your browser with video lessons, coding challenges and screencasts. It has a selection of ‘paths’ you can follow in Ruby, Javascript, HTML & CSS and iOS, or a more advanced Electives path that covers the Git version control system and data visualisation in the R programming language.

[Treehouse](http://teamtreehouse.com/) – Treehouse is a learning website that helps both beginners and professionals develop their coding skills. Although Treehouse works on a paid subscription basis, you can view loads of the videos for free and even take a couple of entire courses without a subscription. Loads of topics are covered including HTML, CSS, Javascript, Ruby, PHP, WordPress and mobile development.

[30 Days to Learn HTML and CSS](https://tutsplus.com/course/30-days-to-learn-html-and-css/) – covering the essentials of HTML and CSS in 30 short videos, this course takes you from the very beginning right through to completing your own website.

[Try Ruby](http://tryruby.org/) – a interactive site that introduces you to the Ruby programming language in 15 minutes. If you like Ruby, then why not move on to:

[Rails for Zombies](http://railsforzombies.org/) – Rails for Zombies lets you learn the Ruby on Rails web application framework directly in your browser, through five zombie-themed videos each followed by exercises. Rails for Zombies may leave you with a craving for brains and entrails. The developers have no liability.

#### Advanced

[Mozilla](https://developer.mozilla.org/en-US/learn) – Mozilla (the developers of Firefox) have put together a whole bunch of resources to help you learn how to make websites as part of their Mozilla Developer Network initiative. Covering HTML, CSS and Javascript, the site includes loads of documentation and technical info for those who are already comfortable with the basics of web programming.

[Google Code University](https://developers.google.com/university/) – Google have put together a wide range of short courses covering more advanced and specialised subjects, including (as you’d expect) Android development and Google API integration, but also more general courses on Ajax, PHP and Go.

[30 Days to Learn jQuery](https://tutsplus.com/course/30-days-to-learn-jquery/) – following on from the 30 Days to Learn HTML and CSS course, this course uses the same format of 30 short videos to introduce you to the jQuery JavaScript library.

[Ruby on Rails Tutorial](http://ruby.railstutorial.org/ruby-on-rails-tutorial-book) – Ruby on Rails Tutorial is a free online book that contains some pretty indepth technical documentation, code examples and code challenges in the Ruby on Rails web application framework.

[Vim Adventures](http://vim-adventures.com/) – Vim is a super useful text editor with loads of powerful features, but it often takes a while to get the hang of. Vim Adventures is an online “Zelda meets text editing” game that uses Vim’s keyboard shortcuts to help you learn Vim – it’ll soon become your new favourite text editor!

[The Github Guide to Git](https://help.github.com/) – Git is a hugely popular version control system that helps you to work in groups and keep control of complex coding projects. Github is a site that hosts your Git repositories (or ‘repos’), and they’ve put together this handy help portal: the Bootcamp section is particularly useful for getting a Git repo setup for your project.
